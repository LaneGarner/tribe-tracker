# Badge Image Generation Guide

Use DALL-E 3 in ChatGPT to generate badge images for Tribe Tracker.

## Style Guidelines

- **Format**: Square 1:1, hexagonal badge shape
- **Style**: 3D achievement badge like Apple Fitness or Garmin Connect
- **Frame**: Metallic silver beveled frame
- **Interior**: Rich gradients, polished enamel-like 3D appearance
- **Background**: Dark background
- **Composition**: Centered, no text (unless badge is a number milestone)

## Workflow

1. Start with the first prompt below
2. For subsequent badges, say "Same style, but with [SUBJECT]"
3. Download each image immediately and name it `{badge_slug}.png`
4. Complete all badges in 1-2 sessions for style consistency

## ChatGPT Plus Limits

- 1 image per request (no batch generation)
- ~40-50 generations per 3-hour window
- Images are 1024x1024 by default
